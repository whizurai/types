# Contributing to @whizurai/types

Thank you for your interest in contributing to @whizurai/types!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/types.git`
3. Install dependencies: `pnpm install`
4. Make your changes
5. Run tests: `pnpm typecheck && pnpm build`
6. Submit a pull request

## Code Style

- Use TypeScript strict mode
- Follow existing code style
- Add JSDoc comments for exported types and interfaces

## Pull Request Process

1. Update documentation if needed
2. Ensure all checks pass
3. Request review from maintainers

For more information, see our [Documentation](https://github.com/whizurai/docs).
